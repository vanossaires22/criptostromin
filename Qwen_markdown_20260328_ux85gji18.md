# 🚀 Crypto Exchange MVP

⚠️ **WARNING:** This is a demo project for educational purposes. DO NOT use for real funds.

## Features
- ✅ Secure Auth (JWT + Bcrypt)
- ✅ Transactional Order Processing (Prisma)
- ✅ Real-time Ready Architecture (WebSocket ready)
- ✅ Dockerized Environment

## Setup
1. `docker-compose up -d`
2. `npm install`
3. `npx prisma migrate dev`
4. `npm run dev`

## Security
- Helmets enabled
- CORS configured
- Input Validation (class-validator)
