# CryptoExchange Pro - Frontend

Современная торговая платформа для криптовалют на HTML/CSS/JavaScript.

## 🚀 Функции

- ✅ Интерактивный график цен (Canvas)
- ✅ Биржевой стакан (Order Book)
- ✅ Форма торговли (Buy/Sell)
- ✅ История сделок
- ✅ Управление ордерами
- ✅ Подключение кошелька
- ✅ Адаптивный дизайн
- ✅ Темная тема
- ✅ Уведомления
- ✅ Real-time обновления

## 📦 Установка

1. Скопируйте все файлы в папку проекта
2. Откройте `index.html` в браузере

## 🛠 Технологии

- HTML5
- CSS3 (CSS Variables, Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- Canvas API для графиков
- Font Awesome для иконок

## 📱 Адаптивность

Платформа полностью адаптирована для:
- Desktop (1200px+)
- Tablet (768px - 1200px)
- Mobile (< 768px)

## ⚠️ Важно

Это **демо-версия** для образовательных целей. Не используйте для реальной торговли!
